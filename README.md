# A Circular Patch Antenna With Double Hybrid Play-pause Slot For Wideband Applications
Circular microstrip patch antenna with double hybrid slot and defected ground structure (DGS) designed on FR-4. Achieves multiband operation with 2.82 GHz wide bandwidth and 6.48 dBi gain. Simulated in Ansys HFSS for C- and X-band applications like satellite communication and radar.
